# EcoCash

Sistema desenvolvido em Java com banco de dados para incentivo à reciclagem por meio de pontuação e recompensas.

---

## Sobre o projeto

O EcoCash é um projeto acadêmico criado para promover a educação ambiental através da tecnologia.  
A plataforma permite que usuários registrem ações de reciclagem, acumulem pontos e acompanhem seu desempenho em rankings, incentivando a prática sustentável de forma gamificada.

---

## Funcionalidades

- Cadastro e autenticação de usuários  
- Registro de ações de reciclagem  
- Cálculo automático de pontos por atividade  
- Consulta de saldo e histórico de pontos  
- Ranking de usuários participantes  
- Resgate de recompensas por pontuação  

---

## Como o cliente utiliza o EcoCash

1. Acessa a plataforma do EcoCash pelo aplicativo ou sistema online  
2. Cria uma conta com nome, e-mail e senha  
3. Realiza login para acessar o sistema  
4. Registra suas reciclagens informando o tipo de material reciclado  
5. Recebe pontos automaticamente a cada registro  
6. Acompanha seu saldo, histórico e posição no ranking  
7. Troca pontos por recompensas disponíveis no sistema  

---

## Tecnologias utilizadas

- Java (JDK)  
- Banco de Dados SQL (MySQL / PostgreSQL)  
- JDBC para integração com o banco de dados  
- SQL para criação e manipulação das tabelas  
- Git e GitHub para versionamento do projeto  
- Postman para testes das APIs  
- Figma para prototipagem das telas
